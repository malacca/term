
## 使用

下载 [term.exe](term.exe)

```
// 获取窗口大小
$ term
143
24

// 无回显输入
$ term pass

// 按键监听
$ term key
```

## 编译

安装 [`MinGW-w64`](https://sourceforge.net/projects/mingw-w64/) 并运行:

```
$ build
```
